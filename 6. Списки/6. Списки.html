Создание и доступ

Список (list) — это упорядоченная изменяемая коллекция объектов произвольных типов.

Создание пустого списка:

list_name = []
list_name = list()

Обращение к элементам списка происходит с помощью оператора список[индекс]. Нумерация начинается с 0.

Массивы можно складывать друг с другом при помощи оператора +.

Общие функции работы с коллекциями

len(список)
Возвращает длину списка.
max(список) / min(список)
Возвращает максимальный / минимальный элемент.
sum(список)
Возвращает сумму элементов (если это числа).

Перебор элементов списка

1 способ: Перебор элементов

lst = [4, 8, 65, 222, 45]
for item in lst:
    print(item)

Данный способ позволяет извлекать элементы последовательно. Обратиться к соседним элементам или изменить сам список сложнее.

2 способ: Перебор по индексам

lst = [4, 8, 65, 222, 45]
for i in range(len(lst)):
    print(lst[i])

При данном подходе имеется возможность обращаться к соседним элементам по индексу (например, i+1).

Генераторы списков

Синтаксическая конструкция для создания списков из других последовательностей.

список = [выражение for переменная in последовательность if условие]

Пример:

# Получить строки длиннее 3 символов
words = ['xxx', 'xxxx', 'xx', 'xxxxx', 'xxxxxx']
lst = [w for w in words if len(w) > 3]
print(lst) # ['xxxx', 'xxxxx', 'xxxxxx']

Срезы

Срезы позволяют получить копию части списка. Результатом среза всегда является список.

список[начало : конец : шаг]

  • начало — индекс начала (включительно). По умолчанию 0.
  • конец — индекс конца (не включительно). По умолчанию длина списка.
  • шаг — шаг извлечения. По умолчанию 1.
nums = [1, 2, 3, 4, 5]

# Разворот списка
print(nums[::-1]) # [5, 4, 3, 2, 1]

# Часть списка
print(nums[2:4]) # [3, 4]

# Элементы на чётных позициях
print(nums[::2]) # [1, 3, 5]

# Элементы на нечётных позициях
print(nums[1::2]) # [2, 4]

Операции со списками

Распаковка списка

lst = [2, 8, 9]
x1, x2, x3 = lst
# x1=2, x2=8, x3=9

Количество переменных должно совпадать с количеством элементов.

Оператор * (звездочка)

nums = [1, 2, 3, 4, 5]
print(*nums) # 1 2 3 4 5

Чтение списка из строки

# Ввод: 45 8 322
nums = list(map(int, input().split()))
print(nums) # [45, 8, 322]

Удаление (del)

lst = [4, 8, 65, 222, 45]
del lst[1] # Удалить элемент с индексом 1
print(lst) # [4, 65, 222, 45]

Двумерные списки

Список, элементами которого являются другие списки (матрица).

Доступ: matrix[строка][столбец]

# Правильная генерация матрицы 10x10, заполненной нулями
matrix = [[0] * 10 for _ in range(10)]

# ОШИБКА: Так делать нельзя (создаются ссылки на один и тот же список)
# matrix = [[0] * 10] * 10

Методы списков

append(значение)
Добавляет элемент в конец списка.
lst = [1, 2]
lst.append(3)
print(lst) # [1, 2, 3]
insert(индекс, значение)
Вставляет элемент на указанную позицию.
lst = [1, 2]
lst.insert(0, 0)
print(lst) # [0, 1, 2]
pop([индекс])
Удаляет элемент по индексу и возвращает его. Если индекс не указан, удаляет последний элемент.
lst = [10, 20, 30]
val = lst.pop() 
print(val) # 30
print(lst) # [10, 20]

val2 = lst.pop(0)
print(val2) # 10
remove(значение)
Удаляет первое вхождение указанного значения. Вызывает ошибку, если элемент не найден.
lst = [1, 2, 3, 2]
lst.remove(2)
print(lst) # [1, 3, 2]
index(значение, [start], [end])
Возвращает индекс первого вхождения значения.
count(значение)
Возвращает количество вхождений значения в списке.
lst = [1, 2, 1, 3, 1]
print(lst.count(1)) # 3
clear()
Удаляет все элементы из списка.
copy()
Создает поверхностную копию списка.
lst = [1, 2, 3]
new_lst = lst.copy()
new_lst[0] = 100
print(lst) # [1, 2, 3]
print(new_lst) # [100, 2, 3]

Сортировка списков

sort(key=..., reverse=...)
Сортирует список на месте (изменяет исходный список).
reverse — сортировка в обратном порядке (по убыванию).
key — функция, которая применяется к каждому элементу перед сравнением.

Примеры использования метода sort()

Обычная сортировка:

lst = [1, 2, 5, 4, 0, 5, 14, 0, 8]
lst.sort()
print(lst) # [0, 0, 1, 2, 4, 5, 5, 8, 14]

Сортировка по убыванию (reverse=True):

lst = [1, 2, 5, 4, 0, 5, 14, 0, 8]
lst.sort(reverse=True)
print(lst) # [14, 8, 5, 5, 4, 2, 1, 0, 0]

Сортировка по ключу (key) - по длине строки:

lst = ['a', 'abba', 'c', 'bb', 's', 'aba', 'bbb']
lst.sort(key=len)
print(lst) # ['a', 'c', 's', 'bb', 'aba', 'bbb', 'abba']

Сортировка по последней букве (lambda):

lst = ['a', 'abbae', 'c', 'bb', 'sd', 'aba', 'bbb']
lst.sort(key=lambda x: x[-1])
print(lst) # ['a', 'aba', 'bb', 'bbb', 'c', 'sd', 'abbae']

Сортировка по нескольким полям:

lst = [['LastName_A', 'FirstName_A'],
       ['LastName_B', 'FirstName_Z'],
       ['LastName_A', 'FirstName_C'],
       ['LastName_B', 'FirstName_B'],
       ['LastName_C', 'FirstName_B'],
       ['LastName_C', 'FirstName_A'],
       ['LastName_A', 'FirstName_B']]
# Сначала по фамилии (0), затем по имени (1)
lst.sort(key=lambda x: (x[0], x[1]))
print(*lst, sep='\n')
# ['LastName_A', 'FirstName_A']
# ['LastName_A', 'FirstName_B']
# ['LastName_A', 'FirstName_C']
# ['LastName_B', 'FirstName_B']
# ...

Функция sorted()

sorted(список, [key], [reverse])
Возвращает новый отсортированный список, не меняя исходный.
lst = [3, 1, 2]
new_lst = sorted(lst)
print(lst) # [3, 1, 2] - остался прежним
print(new_lst) # [1, 2, 3] - новый список
