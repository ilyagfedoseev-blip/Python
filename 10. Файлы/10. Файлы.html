Открытие файлов

Для работы с файлами в Python используется встроенная функция open(), которая возвращает файловый объект.

open(file, mode='r', encoding=None)
Параметры:
file — путь к файлу (абсолютный или относительный).
mode — режим доступа к файлу.
encoding — кодировка (рекомендуется использовать 'utf-8').

Режимы открытия файла (mode):

  • 'r' — чтение (по умолчанию). Файл должен существовать.
  • 'w' — запись. Создает новый файл или перезаписывает существующий.
  • 'x' — эксклюзивное создание. Если файл существует, возникнет ошибка.
  • 'a' — дозапись. Данные добавляются в конец файла.
  • 'b' — двоичный режим (например, 'rb', 'wb').
  • '+' — режим чтения и записи (обновления).
# Пример открытия файла для чтения
f = open('data.txt', 'r', encoding='utf-8')
# ... работа с файлом ...
f.close() # Обязательно закрываем файл!

Чтение из файла

Существует несколько способов прочитать содержимое файла.

read([size])
Читает size символов (или байт). Если size не указан, читает файл целиком.
readline()
Читает одну строку из файла.
readlines()
Читает все строки файла и возвращает их в виде списка.

Пример чтения всего файла

f = open('example.txt', 'r', encoding='utf-8')
content = f.read()
print(content)
f.close()

Запись в файл

Для записи данных используются методы write() и writelines().

write(text)
Записывает строку text в файл. Возвращает количество записанных символов.
writelines(lines)
Записывает список строк lines в файл. Не добавляет переносы строк автоматически!

Пример записи

f = open('output.txt', 'w', encoding='utf-8')
f.write('Привет, мир!\n')
f.write('Это новая строка.')
f.close()

Менеджер контекста with

Конструкция with автоматически закрывает файл после завершения блока кода, даже если возникла ошибка. Это рекомендуемый способ работы с файлами.

with open('file.txt', 'r', encoding='utf-8') as f:
    content = f.read()
    print(content)
# Здесь файл уже закрыт автоматически

Построчное чтение файла

Файловый объект является итерируемым, поэтому его можно перебирать в цикле for. Это эффективно по памяти для больших файлов.

Пример обработки строк

with open('input.txt', 'r', encoding='utf-8') as f:
    for line in f:
        # line содержит строку вместе с символом переноса \n
        clean_line = line.strip() # Удаляем лишние пробелы и \n
        print(clean_line)

Пример из задачи (подсчет суммы чисел в строках)

total_sum = 0
i = 0

# Предполагается, что в input.txt строки вида "10 20 30"
with open('input.txt', 'r', encoding='utf-8') as f:
    for line in f:
        i += 1
        # Разбиваем строку на числа, преобразуем в int и считаем сумму
        numbers = list(map(int, line.split()))
        row_sum = sum(numbers)
        
        total_sum += row_sum
        print(f'Строка {i} - сумма: {row_sum}')

print(f'Общая сумма: {total_sum}')
